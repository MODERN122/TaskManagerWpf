# 💾 Storage Summary (JSON)

## Где хранится файл
- По умолчанию приложение создаёт/обновляет JSON в:
  - `%LocalAppData%\TaskManagerWpf\tasks.json`

## Контракт хранения
- ViewModel работает с `ITaskStore`.
- Реализация: `JsonTaskStore` читает/пишет список `TaskItem` асинхронно.

## Автосохранение
- `MainViewModel` сохраняет изменения “скоро” (debounce), чтобы не писать файл на каждый keypress.
- Перед сохранением делается **snapshot** моделей (`CloneModel()`), чтобы не сохранять VM и не зависеть от UI-коллекции.

## Ошибки
- Ошибки чтения/записи показываются пользователю через `IMessageService` с указанием `StoragePath`.

## Кодировка / русские строки
- В тестах проверяется, что кириллица в JSON пишется без `\uXXXX`-экранирования (UTF-8 текст).

## Рекомендации для изменений
- Любое изменение формата JSON → добавь/обнови тест.
- IO в тестах — только через temp dir, с cleanup.
